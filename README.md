# Task Tracer

Task Tracer is a clean and efficient mobile application built with Flutter, designed to help users organize their daily activities. It allows users to track tasks in a streamlined, user-friendly interface.

## ðŸš€ How to Run the Project

### Prerequisites
* [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
* An Android Emulator, iOS Simulator, or physical device connected.

### Installation Steps
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/chaitanyaa09/task_tracer.git](https://github.com/chaitanyaa09/task_tracer.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd task_tracer
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
